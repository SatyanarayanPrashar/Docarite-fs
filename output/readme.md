# Edloops Quiz

## Description
Edloops Quiz is a web application built using Next.js. It allows users to create and participate in quizzes. The application leverages modern web technologies such as React, TypeScript, and various libraries for state management and styling.

### Technologies Used
- **Next.js**: A React framework for creating server-side rendered and statically generated applications.
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A superset of JavaScript that provides static typing.
- **OpenAI**: For integrating AI functionalities.
- **Lottie**: For rendering animations.
- **Swiper**: For creating responsive sliders.
- **Tailwind CSS**: For styling the application.

This project is private and is intended for educational purposes.

## Getting Started

### Prerequisites
- Node.js (version >= 14)
- npm (Node Package Manager) or Yarn to install dependencies.

### Setting Up the Development Environment

1. Clone the repository:
```bash
git clone https://github.com/yourusername/edloops-quiz.git
```
2. Navigate to the project directory:
```bash
cd edloops-quiz
```
3. Install the dependencies:
```bash
npm install
# or
yarn install
```
4. Run the development server:
```bash
npm run dev
# or
yarn dev
```
5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
 
## Deployment

To deploy your Next.js application, you can use the Vercel platform. Follow these steps:

1. Visit [Vercel](https://vercel.com/) and sign up for an account if you don't have one.
2. After logging in, click on the 'New Project' button.
3. Import your GitHub repository (edloops-quiz).
4. Vercel will automatically detect the Next.js framework and set up the required configurations.
5. Once the deployment is done, you will receive a URL to access your deployed application.

For advanced usage, you can refer to the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying).
